
	1.	Stammdaten eintragen

	•	Oben Name und Empfänger-E-Mail eintragen.
	•	Der Empfängername wird aus der E-Mail automatisch gebildet.

	2.	Rahmenzeiten melden

	•	Tab „Rahmenzeiten“ aktiv.
	•	14/15/16 Stunden wählen.
	•	„Montag bis 18 Uhr“ oder „Mittwoch bis 18 Uhr“ wählen.
	•	Startzeiten pro Tag eingeben: 730 → 07:30 oder direkt 07:30.
	•	Mindestens 5 Tage (Mo–Sa) ausfüllen.
	•	Enter springt zum nächsten Feld.
	•	„E-Mail erstellen“ klicken. Mail prüfen und senden.

	3.	Pause melden

	•	Tab „Pausenzeiten“ öffnen.
	•	Optional „Ort der Pause“ eintippen.
	•	„E-Mail erstellen“ klicken. Mail prüfst du und sendest selbst.

	4.	Speicherung

	•	Nur lokal im Browser (localStorage).
	•	Bleibt beim nächsten Öffnen erhalten.
	•	Reset: Seitendaten im Browser löschen.

	5.	Häufige Fehler

	•	Keine E-Mail eingetragen → Hinweis.
	•	Ungültige Zeit → Hinweis beim betreffenden Tag.
	•	Weniger als 5 Tage → Hinweis, bis genug Tage gefüllt sind.
